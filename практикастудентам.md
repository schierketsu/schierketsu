## О проекте  
Учебный веб-проект для поиска мест практики студентами.
Основная задача — дать простой сценарий:
отфильтровать → посмотреть → выбрать → связаться.
На странице есть карта и список компаний с описанием, стеком, городом и вузами, с которыми они работают.  
Пользователь может зарегистрироваться, оставить отзыв и отправить заявку на практику по e-mail.
Также есть административная часть для управления данными.

### Для студентов
- карта с компаниями  
- фильтры: город, вуз, факультет, технологии  
- просмотр карточек компаний (описание, стек, контакты, галерея, отзывы)  
- отправка заявки на практику по e-mail  
- возможность оставить отзыв  
- личный кабинет  

## Как это работает

Пользователь открывает главную страницу с картой и фильтрами.  
После выбора параметров остаются только подходящие компании.
Далее пользователь открывает карточку компании, изучает информацию и при необходимости:
- отправляет заявку  
- или оставляет отзыв  


## Идея проекта

У студентов разные критерии поиска: город, вуз, факультет, технологии.  
Вместо списка «всё подряд» используется фильтрация и карта, чтобы быстро найти подходящие варианты.
Проект не заменяет HR-систему, а упрощает поиск и первый контакт с компанией.
